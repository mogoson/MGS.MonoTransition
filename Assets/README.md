# MGS.MonoTransition

## Summary

State translation for MonoBehaviour.

## Ability

- UI state transition.
- Transform state transition.
- Renderer state transition.
- Animation state transition.

## Install

- Unity --> Window --> Package Manager --> "+" --> Add package from git URL...

  ```text
  https://github.com/mogoson/MGS.MonoListener.git?path=/Assets
  https://github.com/mogoson/MGS.MonoTransition.git?path=/Assets
  ```
## Samples

- Unity --> Window --> Package Manager --> Packages-Mogoson --> Mono Transition --> Samples.
